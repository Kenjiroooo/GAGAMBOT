# Spider Bot (GAGAMBOT)

Spider Bot is a robotic spider project built using an **Arduino Nano** and **Raspberry Pi Zero 2 W** with an **Raspberry Pi Camera** for smart vision features. The robot is designed with **12 MG90S servos** for leg movement, supports **ball detection**, and can be controlled remotely through a **web interface**. This project combines robotics, embedded systems, and computer vision into a compact multi-legged robot platform.

## Features

- Ball detection using Raspberry Pi Camera
- Web-based remote control
- Multi-leg movement using 12 MG90S servos
- Arduino Nano for servo control
- Raspberry Pi Zero 2 W for processing and camera functions
- Portable power system using 2x 18650 batteries

## Components Used

- 12x MG90S servo motors
- Arduino Nano
- Arduino Nano sensor shield / servo shield
- Raspberry Pi Zero 2 W
- Raspberry Pi Camera Module
- 2x 18650 3.7V batteries
- Buck converter
- Spider robot chassis / frame
- Jumper wires and connectors
- Power wiring setup

## How It Works

The **Arduino Nano** handles the movement of the 12 servo motors, which control the legs of the spider bot. The **Raspberry Pi Zero 2 W** processes the camera feed from the **RPi Cam** for ball detection and hosts the web control interface. The power system uses **two 18650 batteries** with a **buck converter** to provide the required voltage for the electronics.
